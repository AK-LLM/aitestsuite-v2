# aitestsuite-v2

**A next-gen, future-proof AI security and context evaluation suite for LLMs.**

- Red team and blue team ready
- Modular and plug-in friendly
- Demo and production “live” modes
- Streamlit UI + CLI + API ready
- Portable, extensible, and research-grade

## 🚀 Folders

- `core/`: Test engines and API logic (no UI code)
- `scenarios/`: Scenario and prompt bank JSONs
- `ui/`: Streamlit and CLI runner
- `plugins/`: Extensions and custom modules
- `reports/`: Auto-generated reports (PDF, HTML, CSV)
- `docs/`: Guides and documentation
- `tests/`: Automated test scripts

See `/docs/README.md` for more!
